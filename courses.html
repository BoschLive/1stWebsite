Courses

Computer Science I:
-The First course in the Computer Science Degree Map, this course will cover the basics of Computer Science and C++ Programming. Topics Include: Computer Memory, Functions, Basic Algorithms, Structures, Pointers, Classes, etc.
Computer Science II:
-Our CS II course offers a deeper dive into the topics covered in Computer Science I and slightly more advanced C++ Programming. Topics Include: Pointers, Objects, Classes, ADT's, GUI's, etc.
Object Oriented Programming:
-In this course, Students will Begin to examine the difference between Functional and Object-Oriented programs, and use the Java Language to create Objects and Classes
